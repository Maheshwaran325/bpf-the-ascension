# Phase 2 Asset Pack

All assets in this folder were generated in-repo for BPF: The Ascension Phase 2.

- `bg/`: Themed backgrounds per level and menu
- `sprites/`: Player, hazards, enemies, boss elements, and markers
- `ui/`: HUD, modal, button, and card skins
- `audio/music/`: Short loopable theme stubs per level and menu
- `audio/sfx/`: UI and gameplay sound effects

No external attribution is required for this generated pack.
